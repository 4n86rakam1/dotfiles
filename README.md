# dotfiles

## Usage

### Ubuntu 19.10 (eoan)

Run all script within `bootstrap/` directory with the following commnad:

``` bash
wget -qO - https://raw.githubusercontent.com/maya2250/dotfiles/master/install | bash
```

### Windows 10

Run `.\win10\bootstrap.ps1` powershell script.
